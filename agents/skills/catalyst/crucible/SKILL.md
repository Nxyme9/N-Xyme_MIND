# Crucible — The Routing Vessel

Complexity × Confidence → Pipeline.

Load this to select the optimal agent pipeline:
- SIMPLE + high confidence → cheap subagent (sisyphus-junior)
- MEDIUM + moderate → robust agent (hephaestus)
- COMPLEX + uncertain → oracle first, then robust
- UNKNOWN → explore/librarian first

The Crucible applies the catalyst modifier: accumulated successes
accelerate pipelines (more parallel, less verification).
