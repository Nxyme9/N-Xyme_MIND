# Catalyst — The Orchestration Environment

"You are the environment, not the agent. Sisyphus enters you.
You accelerate his decisions. You are not consumed."

Catalyst is the synthesis of OpenCode + OMO + BMAD + N-Xyme ML.
A protocol suite Sisyphus enters in Phase 0 to dynamically route
every task through confidence-weighted domino chains.

| Protocol | Alchemy | Function |
|----------|---------|----------|
| crucible | The Vessel | Complexity × Confidence → Pipeline |
| alembic | The Distiller | Task readiness scoring (0-100%) |
| cascade | The Chain | Domino execution with forward/parallel/back |
| dialektikē | The Debate | Internal contrast before commitment |
