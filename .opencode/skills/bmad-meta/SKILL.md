## What I am

Category gateway for **meta** workflows.
I organize multiple sub-workflows so you pick only what you need.

## Sub-workflows

- **`bmad-help`** — Analyze state and recommend next step
- **`bmad-index-docs`** — Create/update folder index.md
- **`bmad-memory-consolidate`** — Consolidate session to unified memory
- **`bmad-memory-recall`** — Retrieve relevant context
- **`bmad-memory-search`** — Find across all memory sources
- **`bmad-party-mode`** — Multi-agent group discussion
- **`bmad-distillator`** — Lossless LLM-optimized compression
- **`bmad-advanced-elicitation`** — Push LLM to reconsider and refine
- **`bmad-document-project`** — Brownfield project documentation
- **`bmad-shard-doc`** — Split large markdown into files
- **`bmad-catalyst-orchestration`** — Multi-agent fractal delegation
- **`bmad-generate-project-context`** — Generate project-context.md
- **`bmad-sprint-planning`** — Generate sprint from epics
- **`bmad-sprint-status`** — Summarize sprint and surface risks
