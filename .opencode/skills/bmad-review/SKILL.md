## What I am

Category gateway for **review** workflows.
I organize multiple sub-workflows so you pick only what you need.

## Sub-workflows

- **`bmad-code-review`** — Parallel review: Blind Hunter, Edge Case, Acceptance Auditor
- **`bmad-review-adversarial-general`** — Cynical findings report
- **`bmad-review-edge-case-hunter`** — Exhaustive boundary condition analysis
- **`bmad-editorial-review-prose`** — Clinical copy-edit for communication
- **`bmad-editorial-review-structure`** — Cut, reorganize, simplify
- **`bmad-retrospective`** — Post-epic lessons and assessment

## How to use

1. Review the sub-workflows above
2. Identify which one matches the user's need
3. The full workflow instructions are in `bmad/` — read the relevant `workflow.md` file:
      - `bmad/.../bmad-code-review/workflow.md`
   - `bmad/.../bmad-review-adversarial-general/workflow.md`
   - `bmad/.../bmad-review-edge-case-hunter/workflow.md`
   - `bmad/.../bmad-editorial-review-prose/workflow.md`
   - `bmad/.../bmad-editorial-review-structure/workflow.md`
   - `bmad/.../bmad-retrospective/workflow.md`
4. Follow the steps in that file
