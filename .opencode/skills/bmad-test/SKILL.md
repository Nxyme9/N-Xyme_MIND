## What I am

Category gateway for **test** workflows.
I organize multiple sub-workflows so you pick only what you need.

## Sub-workflows

- **`bmad-teach-me-testing`** — Progressive testing study sessions
- **`bmad-testarch-atdd`** — Acceptance tests with TDD cycle
- **`bmad-testarch-automate`** — Expand test automation coverage
- **`bmad-testarch-ci`** — CI/CD quality pipeline with test gates
- **`bmad-testarch-framework`** — Initialize Playwright or Cypress
- **`bmad-testarch-nfr`** — Performance, security, reliability
- **`bmad-testarch-test-design`** — System/epic-level test plans
- **`bmad-testarch-test-review`** — Evaluate test quality
- **`bmad-testarch-trace`** — Requirements-to-test matrix
- **`bmad-qa-generate-e2e-tests`** — End-to-end automated tests

## How to use

1. Review the sub-workflows above
2. Identify which one matches the user's need
