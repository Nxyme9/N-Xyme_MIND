# Contributing to N-Xyme MIND

Thanks for your interest! Here's how to contribute.

## Getting Started
1. Fork the repo
2. Clone your fork
3. Run `bash scripts/setup.sh` to install dependencies
4. Create a branch: `git checkout -b feature/my-feature`

## Code Style
- Mojo: follow existing patterns in `services/mojo/src/`
- Python: PEP 8
- Keep functions focused and under 100 lines where possible

## Pull Request Process
1. Update documentation if needed
2. Run `bash scripts/release-check.sh` to verify
3. Create PR using the template
4. One reviewer must approve before merge

## Questions?
Open a Discussion or issue.
