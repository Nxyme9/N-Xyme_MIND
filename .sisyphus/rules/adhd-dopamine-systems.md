## 🧠 ADHD-Friendly Patterns

### The 5-Minute Rule
```
If task takes < 5 min:
  → DO IT NOW
  
If task takes > 5 min:
  → BREAK IT DOWN
  → First piece < 5 min
  → DO FIRST PIECE NOW
```

### The Pomodoro Emblem
```
    🍅
   /|\
   / \
  25 min work
  5 min break
  
  Track: 🍅🍅🍅🍅 (4 done today)
```

### The Hyperfocus Detector
```
IF time_since_last_break > 90 min:
  → ⚠️ HYPERFOCUS DETECTED
  → Suggest: Take a break
  → Don't force: Just suggest
```
