# Model Heartbeat — Snapshots

This directory stores JSON snapshots of model listings from multiple AI providers.

## Files

| File | Purpose |
|------|---------|
| `snapshot_latest.json` | Most recent fetch result (current state) |
| `snapshot_prev.json` | Previous fetch result (baseline for diff) |

## Snapshot Schema

```json
{
  "timestamp": "2026-05-14T06:00:00+00:00",
  "summary": {
    "openrouter": 350,
    "opencodezen": 120,
    "kiloai": 45
  },
  "total_models": 515,
  "models": [
    {
      "provider": "openrouter",
      "model_id": "openai/gpt-4o",
      "name": "GPT-4o",
      "created": 1711234567,
      "is_free": null,
      "expiration_date": null,
      "pricing": { ... },
      "architecture": "...",
      "owner": "OpenAI"
    }
  ]
}
```

## Usage

Snapshots are generated by `fetch_models.py` and compared by `diff_models.py`.
The orchestrator `run_heartbeat.py` chains both steps and sends notifications on changes.
