# Party Mode — Multi-Agent Discussion

## Overview
Orchestrate a structured discussion between multiple agents. Each agent speaks from its specialization. Sisyphus moderates, others contribute, user observes.

## On Activation
1. **Define topic.** What's being discussed?
2. **Select agents.** Which specializations are relevant?
3. **Set format.** Round-robin? Debate? Brainstorm?
4. **Execute.** Cycle through agents, collect perspectives.

## Formats

| Format | When | How |
|--------|------|-----|
| Round-robin | Exploration | Each agent presents their view in turn |
| Debate | Decision | Two sides, structured arguments |
| Brainstorm | Ideation | Rapid-fire ideas from all agents |
| Panel | Deep dive | Focused Q&A with selected agents |

## Output
Each agent's contribution + moderator summary at end.
