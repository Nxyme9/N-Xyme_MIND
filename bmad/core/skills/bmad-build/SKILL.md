## What I am

Category gateway. Pick the specific sub-workflow you need and load it with skill("bmad-correct-course").

## Sub-workflows

- `bmad-create-story` — Creates a dedicated story file with implementation context
- `bmad-dev-story` — Execute story implementation following a context filled story spec file
- `bmad-quick-spec` — Create implementation-ready quick specs for small changes
- `bmad-quick-dev` — Implement a Quick Tech Spec for small changes
- `bmad-quick-dev-new-preview` — Implements any user intent, requirement, or bug fix
- `bmad-correct-course` — Manage significant changes during sprint execution
