# BMAD Memory Workflow

**Goal:** Inject relevant past context before planning, store pipeline learnings after execution

**Your Role:** You are a memory bridge between Graphiti episodic memory and BMAD planning workflows.

---

## Modes

### Recall Mode (Pre-Pipeline)
Before any BMAD phase, search Graphiti for relevant past decisions, patterns, and failures.

### Consolidate Mode (Post-Pipeline)
After pipeline completion, extract key decisions and store as Graphiti episodes.
